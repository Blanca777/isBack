## 实现三栏布局

#### float+margin布局

```html
<style>
    .layout1 .left{
        width: 300px;
        height: 100px;
        background-color: green;
        float: left;
    }
    .layout1 .center{
        height: 100px;
        background-color: blue;
    }
    .layout1 .right{
        width: 300px;
        height: 100px;
        float: right;
        background-color: red;
    }
</style>
<section class="layout1">
    <article>
        <div class="left"></div>
        <div class="right"></div>
        <div class="center"></div>
    </article>
</section>
```

#### position

```html
<style>
    .layout2{
        height: 100px;
        position: relative;
    }
    .layout2 .left{
        position: absolute;
        left: 0;
        width: 300px;
        height: 100px;
        background-color: green;
    }
    .layout2 .center{
        height: 100px;
        background-color: blue;
        margin-left: 300px;
    }
    .layout2 .right{
        position: absolute;
        top: 0;
        right: 0;
        width: 300px;
        height: 100px;
        background-color: red;
    }
</style>
<section class="layout2">
    <article>
        <div class="left"></div>
        <div class="center"></div>
        <div class="right"></div>
    </article>
</section>
```

#### flexbox布局

```html
 <style>
     .layout3 article{
         display: flex;
     }
     .layout3 .left{
         width: 300px;
         height: 100px;
         background-color: green;
     }
     .layout3 .center{
         flex: 1;
         background-color: blue;
     }
     .layout3 .right{
         width: 300px;
         height: 100px;
         background-color: red;
     }
</style>
<section class="layout3">
    <article>
        <div class="left"></div>
        <div class="center"></div>
        <div class="right"></div>

    </article>
</section>
```

#### table

```html
<style>
    .layout4 article{
        display: table;
        width: 100%;
        height: 100px;
    }
    .layout4 .left{
        display: table-cell;
        width: 300px;
        background-color: green;
    }
    .layout4 .center{
        display: table-cell;
        background-color: blue;

    }
    .layout4 .right{
        display: table-cell;
        width: 300px;
        background-color: red;
    }
</style>
<section class="layout4">
    <article>
        <div class="left"></div>
        <div class="center"></div>
        <div class="right"></div><style>
            .layout4 article{
                display: table;
                width: 100%;
                height: 100px;
            }
            .layout4 .left{
                display: table-cell;
                width: 300px;
                background-color: green;
            }
            .layout4 .center{
                display: table-cell;
                margin-left: 300px;
                background-color: blue;
                
            }
            .layout4 .right{
                display: table-cell;
                width: 300px;
                background-color: red;

            }
        </style>
    </article>
</section>
```
